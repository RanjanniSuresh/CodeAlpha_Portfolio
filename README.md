# Simple Calculator

This is a basic calculator implemented with HTML, CSS and JavaScript.

Features:
- Basic arithmetic: addition, subtraction, multiplication, division
- Percent support (e.g., `50%` -> `0.5`)
- Clear (`C`) and delete (`DEL`) buttons
- Real-time result preview while typing
- Keyboard support for numbers, `+ - * /`, `Enter` for equals, `Backspace`, `Escape` to clear

How to run:
1. Open `index.html` in your browser (double-click or `File -> Open`).
2. Use buttons or keyboard to perform calculations.
